## 0. 한 줄 요약 — 세 진영

| 진영 | 특징 | 한계 |
|---|---|---|
| 엣지 깜빡임 (Carcellar 2024, Reddy 2019) | 저비용 Pi, 실시간 지향 | 횟수만·배치·저정확, **강건화 없음** |
| 강건/정밀 검출 (InstBlink 2023, 야간운전 파이프라인 2025) | in-the-wild·저조도 보정 | **GPU급이거나 엣지 아님** |

## 1. 문제정의

실시간 깜빡임 검출은 엣지에서 (i) 단순 횟수·배치·저정확에 머물거나, (ii) 강건한 최신 기법은 GPU급/비엣지다. 특히 **저조도·저해상도에서 랜드마크 검출이 급격히 무너지는데, 이를 엣지에서 실시간으로 보완하는 시스템이 없다.**

> "라즈베리파이5에서, STD 기반 실시간 깜빡임 검출(횟수/BPM)을 하되, 저조도·저해상도 환경에서도 경량 강건화(SR/저조도 보정)로 검출을 유지하는 온디바이스 시스템이 없다."

- **기여 두 축:** (a) STD 4단계 + 개인 캘리브 기반 **실시간 강건 깜빡임 검출**, (b) **엣지용 경량 강건화 전처리**로 "어떤 환경에서도" + fps·전력 실측.
- **필요성:** 저전력·상시·로컬 처리(프라이버시)·저지연.

## 2. 관련연구 (그룹)

- **A. EAR·실시간 검출:** Soukupová & Čech 2016(EAR 정의, SVM/HMM), Savitzky–Golay 강건화 2018, **InstBlink/MPEblink 2023**(다인원 실시간 SOTA, GPU급).
- **B. 상태머신·4단계:** FSM 전이 기반 판정, 4단계(closing/closed/opening) 지속 100–400ms, Kwon 2013 운동학(닫힘>열림, ~0.5초).
- **C. 적응형 임계값:** adaptive EAR, HMM 개인 적응 → 우리는 **EAR 이동평균 개인 캘리브**.
- **D. 엣지 baseline:** Carcellar 2024(Pi3B+, 85%, 20초 배치), Reddy 2019(83%), **야간운전 저조도+EAR 파이프라인 2025**.
- **E. 강건화:** SR(Super-FAN/EfficientSRFace — 포화, ESPCN/FSRCNN/edge-SR — 경량, 순차 손해 주의), 저조도(**Zero-DCE** 경량).
- **F. 플랫폼:** MediaPipe FaceLandmarker ARM CPU 27–30fps.

## 3. 한계점 정의

- **Carcellar 2024:** 횟수만·20초 배치·정상 재현율 ~44%·유효 ~8fps·**강건화 전무**.
- **InstBlink 등 강건 기법:** GPU급 → 엣지 실시간 아님.
- **야간운전 파이프라인 2025:** 저조도+EAR 아이디어가 가장 근접하나 운전 응용·엣지 Pi 아님.
- **SR–랜드마크:** 이미 포화 + 순차(SR→검출) 손해 리스크.

## 4. 내 개선점 (한계 → 대응)

- **STD 4단계 + 개인 캘리브(이동평균):** 프레임별 오검출(1초 폐안=30회) 방지, 실시간 강건 카운트.
- **배치 → 실시간:** Carcellar의 20초 배치 대비 상시 실시간.
- **엣지 온디바이스 강건화:** 저조도/저해상도에서 경량 SR/저조도 보정 → "어떤 환경에서도" (핵심 기여).
- **FPS 스윕:** 60/30/15로 최소 사양 규명(15fps 충분한가).
- **서버-클라이언트(Flask MJPEG):** 카메라 위치 무관 이식성.

## 5. 평가·비교 방식 (예정)

- **데이터:** 공개셋(Eyeblink8/ZJU/RT-BENE/mEBAL2) + 2인 30초 큐 기반 정답.
- **검출 성능:** 인식률(정답 대비), 가능하면 F1 — 공개셋 baseline과 비교.
- **FPS 스윕:** 60/30/15에서 정확도·실시간성 유지 여부.
- **강건화 실험:** 공개셋 다운스케일/암전 테스트셋에서 SR/저조도 유무 검출 회복 + **Pi5 fps·전력·모델크기** 실측.

## 7. 리스크 / 확인

- **SR/저조도 보정이 실제로 검출을 돕는가** — 순차 손해 사례 → 초기 검증.
- **Pi5 경량 강건화 실시간(~5fps) 실측** — 되는 배율·모델 확인.
- **강건화 신규성** — 포화 회피 = "엣지 + 실시간 + 깜빡임 응용" 조합으로 차별화.
- **IRB 필요 여부** — 일단 2인 내부 실험으로 완성 우선.
