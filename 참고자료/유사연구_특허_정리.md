# 유사 논문·특허 정리 (유사성 높은 것 위주) — 박사님용

정리일: 2026-07-17
우리 방향: 라즈베리파이5 실시간 깜빡임 검출(EAR + STD) + **눈 영역 품질 기반 조건부 경량 SR** + **개인별 EAR 캘리브**

각 항목에 "유사성"과 "우리와의 차별점"을 같이 적었습니다(신규성 방어용).

## 1. 관련 논문 (유사성 높은 것 3개)

### P1. Carcellar, Tychuaco, Yumang — Self-Applicable Eye Strain Detection… Raspberry Pi (IEEE IICAIET 2024)
- 내용: YOLOv5, Raspberry Pi 3B+, 20초 영상으로 깜빡임률 산출 → 눈 피로, 정확도 85%.
- **유사성:** 엣지(Pi)에서 깜빡임 검출. 우리 baseline.
- **차별점:** 우리는 (a) EAR+STD 경량 로직(무거운 검출망 아님), (b) 개인별 캘리브, (c) 눈 품질 기반 **조건부 SR**을 결합. 그들은 20초 배치·저정확·SR/개인화 없음.
- 링크: https://ieeexplore.ieee.org/document/10730733

### P2. Improving Driver Drowsiness Detection via Personalized EAR/MAR Thresholds and CNN (arXiv, 2025)
- 내용: 사람마다 눈/입 모양이 달라 고정 임계값이 부정확 → **피험자별 EAR/MAR 임계값 개인화** + CNN 분류.
- **유사성:** ★ 우리 "개인별 EAR 캘리브" 근거와 직결. "고정 threshold로는 제대로 평가 안 된다"는 박사님 논리를 그대로 뒷받침.
- **차별점:** 운전 졸음·비엣지·CNN 중심. 우리는 **엣지 실시간 + STD 판정 + 조건부 SR**과 결합하고, 캘리브 절차를 4인 반복 실험으로 통계 검증.
- 링크: https://arxiv.org/abs/2604.22479

### P3. Nighttime Driver Fatigue — 저조도 향상 + 얼굴/랜드마크 + EAR (Modelling, 2025)
- 내용: 저조도 보정(Zero-DCE 등) → 얼굴·랜드마크 검출 → EAR → 피로 판정. **"화질 전처리 → 랜드마크 → EAR → 깜빡임"** 파이프라인.
- **유사성:** ★ 우리 파이프라인 구조(품질 전처리 후 랜드마크·EAR)와 가장 유사.
- **차별점:** 저조도(조도) 향상 vs 우리 해상도(SR), 운전·비엣지, 무조건 전처리 vs 우리 **눈 픽셀 크기 기반 조건부 게이트**(효율).
- (SR 필요성 보조 근거) 저해상도 <30×30에서 랜드마크 오차 급증: arXiv 1907.13255 / CLAHE+SR로 눈 검출 개선: PMC12273955(2025).

## 2. 관련 특허 (유사성 높은 것 3개)

### 특허1. US 9,690,988 B2 — Image processing apparatus/method for blink detection in an image (등록)
- 내용: 영상에서 **깜빡임 검출**, 그리고 **저해상도 눈 영역의 SR 처리**를 함께 다룸(감은 눈 프레임은 SR에 쓰지 않는 등).
- **유사성:** ★ "깜빡임 검출 + 저해상도 눈 영역 + SR"을 결합 — 우리 조건부 SR+깜빡임과 개념적으로 가장 가까움.
- **차별점:** 우리는 "**눈 픽셀 품질로 SR을 켤지 말지 결정하는 조건부 게이트 + 엣지 실시간 + EAR/STD 상태머신 판정**"으로 차별. (정독 후 청구항 대비 필요)
- 링크: https://patents.google.com/patent/US9690988B2/en

### 특허2. US 11,861,916 B2 — Driver alertness monitoring system (2024, 등록)
- 내용: IR로 운전자 각성 모니터, **눈꺼풀 닫힘/열림 속도** 사용.
- **유사성:** 눈꺼풀 개폐 속도 기반 깜빡임/각성 판단.
- **차별점:** IR·운전·SR 없음. 우리는 일반 RGB 웹캠 + 조건부 SR + 엣지 실시간.
- 링크: https://patents.justia.com/patent/11861916

### 특허3. US 2021/0282669 A1 — 안경 IR 기반 깜빡임 역학 측정 (공개/출원)
- 내용: 안경 IR LED/검출기로 완전/불완전 여부·닫힘/열림 속도·눈꺼풀 속도 측정.
- **유사성:** 깜빡임 역학·완성도(우리 completeness 옵션과 연관).
- **차별점:** IR 웨어러블. 우리는 카메라 + 엣지 SR + STD.
- 링크: https://patents.google.com/patent/US20210282669A1/en

## 3. 한 줄 신규성 정리

> 개별 요소(엣지 깜빡임 검출, 개인별 EAR 임계값, SR로 저해상도 얼굴/눈 보정)는 각각 존재하지만,
> **"눈 영역 픽셀 품질에 따른 조건부 경량 SR(라즈베리파이5 실시간) + 개인별 EAR 캘리브 + STD 상태머신 판정"의 결합**은 비어 있음.
> 특히 (1) 무조건이 아니라 **눈 픽셀 기준으로 SR을 켜고 끄는 판단 로직**(효율=엣지 경량화), (2) 엣지 실시간 실측(fps/전력)이 우리 method의 핵심.

※ 특허 등록/공개 상태·청구항은 게재/출원본(Google Patents·KIPRIS)에서 최종 확인 권장.
