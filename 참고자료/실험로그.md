# 실험 로그 (Experiment Log)

## 2026-07-17 — 엣지 실현성 1차 (bring-up)

**구성:** 맥북 웹캠(Logitech C980) → MJPEG(Flask) 서버 → 라즈베리파이5 클라이언트
→ MediaPipe FaceLandmarker(VIDEO 모드) → EAR → STD 4단계 상태머신 → 카운트.
Pi 환경: conda(miniforge) Python 3.11 + mediapipe 0.10.18 + opencv-python 4.11 + flask.

**결과:**
- FaceLandmarker 처리시간: **mean 24.2 ms/frame, p95 26.3 ms → 이론상 max ~41.2 fps** (3164 프레임 측정)
- 파이프라인 실측: **capture ~25.0 fps** (입력 스트림/오버헤드 제한)
- 깜빡임: 28회 검출, bpm 23.0, 상태머신 정상 동작. completeness 대부분 1.00, 1건 0.00(불완전으로 판정).

**결론:**
- **엣지 실시간 실현성 확인.** "15fps 충분한가?"는 여유롭게 충족(25fps 실측, 랜드마커 단독 41fps 가능).
- 25fps 프레임예산(40ms) 중 랜드마커 24ms → **~16ms headroom** → 경량 SR/저조도 전처리를 얹어도 실시간 유지 가능성 → 핵심 기여의 실현 가능성 시사.

**다음:**

- 2인 30초 실험 + 큐 기반 정답 → 인식 정확도.
- SR/저조도 강건화 전처리 실험(다운스케일/암전 셋).
